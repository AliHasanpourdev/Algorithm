# Algorithm

مجموعه پیاده‌سازی الگوریتم‌های پایه‌ای (مرتب‌سازی، جستجو و ...) به زبان Python.

## محتوا
- Binary Search و تست آن
- الگوریتم‌های مرتب‌سازی
- الگوریتم‌های جستجو
- تست الگوریتم‌ها (`test_algorithm`)

## هدف
تمرین Problem Solving و آماده‌سازی برای مصاحبه‌های فنی.

## نحوه اجرا

```bash
python Binery_Search.py
```

**در حال تکمیل** — الگوریتم‌های بیشتری اضافه خواهد شد.
